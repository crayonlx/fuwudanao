# 如何内嵌代码片段

云雀支持原始 Markdown 代码格式显示。如下：

```
<div> Hello </div>

```


如果想要代码演示效果，可以使用 [Riddle](http://riddle.alibaba-inc.com/) 。在 Riddle 中点击`复制链接图标`，点击 `内嵌到外部页面中`，在弹出的对话框中选择 `iframe ` 方式,将代码贴到云雀文档中即可。
