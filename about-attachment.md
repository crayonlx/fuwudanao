# 附件支持哪些格式

> 注意：云雀不是一个文件系统，所以请慎重选择要上传到云雀的附件。我们推荐在云雀的文档中，贴入已有的附件链接

那么，我们支持哪些附件格式呢？

```
// image
'.jpg', '.jpeg', 	
'.png', 					
'.gif', 					
'.bmp', 					
'.wbmp', 					
'.webp',
'.tif',
'.psd',
'.sketch',
// text
 '.svg',
 '.js', '.jsx',
 '.json',
 '.css', '.less',
 '.html', '.htm',
 '.xml',
// tar
 '.zip',
 '.gz', '.tgz', '.gzip',
// video
 '.mp3',
 '.mp4',
 '.avi',
// office
 '.doc', '.docx', '.page',
 '.ppt', '.pptx', '.key',
 '.xlsx', 
 '.pdf',
 '.md',
 
```
